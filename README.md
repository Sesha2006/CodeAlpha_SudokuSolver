# CodeAlpha Sudoku Solver

## Introduction
This project was created as part of the CodeAlpha C++ Programming Internship.

The program solves a Sudoku puzzle using the Backtracking Algorithm. It checks whether a number can be placed in a particular position by following Sudoku rules and recursively solves the puzzle until the complete solution is found.

---

## Features
- Uses a 2D array to represent the Sudoku board
- Implements recursive backtracking
- Validates rows, columns, and 3x3 grids
- Displays solved Sudoku puzzle clearly

---

## Technologies Used
- C++
- Recursion
- Backtracking Algorithm
